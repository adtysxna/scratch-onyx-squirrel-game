# Onyx Catch Game 

This is my Week 0 project for CS50.

## Description

A simple Scratch game where Onyx (the cat) follows the mouse and catches squirrels to increase score within a time limit.

## How to Play

* Move the mouse to control Onyx
* Catch as many squirrels as possible before time runs out
